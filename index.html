Prework Study Guide

HTML

  • HyperText Markup Language - structure and content of web pages.
  • Uses tags like h1, p, a, for text and links.
    • Basic structure: !DOCTYPE, html, head, body.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Cascading Style Sheets - design and layout of web pages.
  • Selectors target HTML elements to style.
  • Properties define color, size, font, and more.
  • Values specify styling details like colors and sizes.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Version control system - track changes in code projects.
  • Git commands: clone, add, commit, push, pull.
  • Repositories store project history.
  • Git branching for parallel development.
  • Collaborate using Git with platforms like GitHub or GitLab.

JavaScript

  • Scripting language - adds interactivity to web pages.
  • Variables store data (e.g., var x = 5;).
  • Functions perform tasks (e.g., function myFunction() {...}).
  • Conditional statements (if, else) control program flow.
  • DOM manipulation for dynamic web content updates.
